# Bitebook - Digital Recipe Book

**Role:** Full-Stack Developer  
**Team:** Team of 2
**Tech:** JavaScript, HTML, CSS, PHP, MySQL  

## Project Overview
Bitebook is a web application designed for organizing, storing, and filtering recipes.  
Users can create profiles, save recipes, and search by ingredients. The app focuses on a responsive and user-friendly interface.

## Features
- User registration, login, and session management  
- Recipe creation, editing, and deletion (CRUD)  
- Filtering recipes by ingredients   
- Secure server-side logic with PHP and MySQL
